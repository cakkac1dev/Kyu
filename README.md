# Kyu

**Неофициальный десктоп-клиент для AniXart (Windows)**

Смотри аниме прямо в окне, со встроенным адблокером, веди списки и синхронизируйся с аккаунтом AniXart.

</div>

---

> ⚠️ **Дисклеймер.** Kyu — это неофициальный фан-клиент **для** AniXart, а не официальное приложение.
> Проект **не связан** с командой AniXart и сделан энтузиастом. Все данные берутся из публичного API AniXart.
> Сделано с ❤️ вайбиком и claude-bro.

## Возможности

- 🛡️ **Встроенный адблокер** — режет рекламу в плеерах Kodik и Sibnet
- 📺 **Просмотр в окне** — выбор озвучки и источника, полноэкранный режим
- 🗂️ **Каталог и поиск** — по названию и по пользователям
- 📑 **Закладки** — Смотрю / В планах / Просмотрено / Отложено / Брошено / Избранное / Коллекции
- 🔄 **Синхронизация** с аккаунтом AniXart — прогресс, списки, история
- 📊 **Профиль со статистикой** — графики, кольцевая диаграмма, время просмотра, друзья
- 📡 **Лента и каналы** — подписки, статьи, комментарии, оценки
- 🎬 Отметки просмотренных серий и продолжение с нужной серии
- 🐱 Тёмная тема, анимации, своя навигация и управление окном

## Запуск (из исходников)

Нужны [Node.js](https://nodejs.org) 18+ и npm.

```bash
npm install        # установит Electron и зависимости
npm start          # запустит приложение
```

> Если бинарник Electron не скачался автоматически (политика npm `allow-scripts`):
> `node node_modules/electron/install.js`

## Сборка

```bash
npm run dist   # соберёт установщик + ZIP в dist/
```

Создаются **`Kyu-Setup-<версия>.exe`** (NSIS-установщик) и **`Kyu-<версия>-win.zip`** (переносимая папка).
Иконка и метаданные берутся из `build/icon.png` и `package.json`.

## Куда сохраняются данные

Аккаунт (токен сессии), избранное и история хранятся **локально** в
`%APPDATA%/Kyu/data/store.json`. В репозитории этих данных нет. Пароль нигде не сохраняется —
он уходит только на сервер AniXart при входе.

## Структура

| Путь | Назначение |
|------|-----------|
| `main.js` | Главный процесс Electron: окно, прокси к API, адблокер, реферер плееров, IPC |
| `preload.js` | Безопасный мост `window.anixart` (contextBridge) |
| `src/` | Интерфейс: `index.html`, `renderer.js`, `styles.css`, плеер-обёртка `player.html`, мост `playerbridge.js`, иконка |
| `build/icon.png` | Иконка для сборки exe |

Подробно об устройстве — в [ARCHITECTURE.md](ARCHITECTURE.md).
